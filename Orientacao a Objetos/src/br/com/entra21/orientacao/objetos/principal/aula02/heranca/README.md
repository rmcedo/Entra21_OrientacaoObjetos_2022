# Aprendendo sobre HERANÇA

## Herança é a abstração para aproveitar atributos e métodos de uma outra classe

> Essa classe que está provendo atributos e métodos, podemos chama-lá de super Classe.

> Para realizar essa herança em uma sub classe temos que usar a palavra reservada **extends**

> Quando uma sub classe herda atributos de métodos da super Classe não é mais necessário manter meus atributos e métodos escritos na minha sub classe.
